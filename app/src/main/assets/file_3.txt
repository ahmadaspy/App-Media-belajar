HTML Formatting Elements
didalam HTML kita mendifinisikan kata dengan sesuatu yang spesial
apakah kata tersebut penting atau perlu digaris bawahi
special type seperti penggunaan bold <b> atau italic <i> untuk sebuah paragraph atau kata
di dalam HTML.

formatingnya sebagai berikut :
<b> - Bold text
<strong> - Important text
<i> - Italic text
<em> - Emphasized text
<mark> - Marked text
<small> - Small text
<del> - Deleted text
<ins> - Inserted text
<sub> - Subscript text
<sup> - Superscript text

coba koding dibawah ini !
<!DOCTYPE html>
<html>
<body>

<p>This text is normal.</p>
<p><strong>This text is strong.</strong></p>
<p><b>This text is bold</b></p>
<p><i>This text is italic</i></p>
<p><em>This text is emphasized</em></p>
<p><h2>HTML <small>Small</small> Formatting</h2></p>
<p><h2>HTML <mark>Marked</mark> Formatting</h2>
<p>My favorite color is <del>blue</del> red.</p>
<p>My favorite <ins>color</ins> is red.</p>
<p>This is <sub>subscripted</sub> text.</p>
<p>This is <sup>superscripted</sup> text.</p>

</body>
</html>
